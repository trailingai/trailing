# YC Batch Thesis Analysis (S25 → W26 → X26)

## Batch evolution

### S25 (Summer 2025)
- ~160 companies, 88% AI-native
- Agent saturation: ~50% vertical agents
- Peak "everyone builds agents" era

### F25 (Fall 2025)
- ~100 companies
- Agent concentration dropped to 15-17%
- Diversification into content/video/data
- YC got bored of generic agents

### W26 (Winter 2026)
- ~200 companies (largest batch)
- Agents retreated to 19%
- Hardware surge: 18 companies
- Research/data infrastructure grew to 10%
- 14 companies at $1M+ ARR before Demo Day (3x W25)
- 14% average weekly growth across batch
- 56/198 fully autonomous agent companies
- Healthcare: 22 companies (largest vertical)
- 24 repeat founders in batch

### X26 (Spring 2026)
- ~140+ companies
- Agents rebounded to 30-35% (but infrastructure-focused, not generic)
- Infrastructure focus grew to 20%

## W26 Winner Patterns

### By traction type
- Luel: 190K contributors, thousands signups/day (training data)
- Didit: Millions verified (identity for agents)
- Pax Historia: 35K DAUs (consumer AI)
- Pocket: $27M ARR, 30K units, 50% MoM (hardware)
- Mango Medical: $10M ARR LOI (vertical agent)
- Samora AI: 1M+ calls processed (deployed agent)

### Winning categories
1. Vertical AI agents in regulated industries (healthcare, legal, finance, construction)
2. Agent infrastructure (guardrails, monitoring, identity, memory, benchmarks)
3. Physical-world AI (robotics, drones, hardware)
4. Training data at consumer scale
5. Consumer AI with genuine engagement (35K+ DAUs)

### Non-winners
- Generic "AI-powered SaaS"
- Horizontal agent platforms
- Fintech for consumers (crowded + regulatory)
- Pure research without application

## Partner Specializations
- Brad Flora: 43% fintech infrastructure
- Ankit Gupta: Physical-world automation (only partner with serious hardware thesis)
- Tyler Bosmeny: Healthcare + defense + space (most diversified, 30 companies)
- Diana Hu: Healthcare + agent infrastructure (23 companies)
- Jared Friedman: Unsexy essential infra for auditable/legal AI (30 companies)
- Gustaf Alstromer: Regulated industries (26 companies)

## Implication for Trailing
Trailing fits into Category 2 (agent infrastructure) with strong overlap into Category 1 (regulated industries). The risk is being perceived as a "horizontal agent platform" (non-winner category). The fix: go vertical — pick one regulated industry and build compliance specifically for that domain.
